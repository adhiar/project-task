
# Discount Calculator

## Requirements
- Python 3.x

## Dependencies
- `tkinter` (for GUI)

### How to Install tkinter
Tkinter is included with most Python installations. If you need to install it manually:

- **Windows/Linux:**
	Tkinter is usually included by default. If not, you can install it using:
	```sh
	pip install tk
	```
- **Ubuntu/Debian:**
	```sh
	sudo apt-get install python3-tk
	```

Once installed, you can run `cal.py` to use the calculator.
