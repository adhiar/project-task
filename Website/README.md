# Marvel Heroes Website

## How to View Locally

1. Make sure all files (HTML, CSS, images) are in the `Website` folder.
2. Open `index.html` with your web browser (double-click or right-click and choose "Open with").
3. The website will display locally on your computer.

**Note:** An internet connection is required to load Bootstrap from the CDN. If you are offline, some styles may not appear correctly.

No server or extra setup is required.
